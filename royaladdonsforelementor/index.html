1. What is Royal Addons for Elementor?১. রয়্যাল অ্যাডন্স ফর এলিমেন্টর কী?

Royal Addons for Elementor extends the Elementor page builder with 130+ custom widgets. Instead of writing code, simply drag and drop sliders, carousels, navigation menus, post grids, WooCommerce displays, social feeds, charts, and more onto any page. Think of Elementor as a canvas and Royal Addons as a massive professional toolkit of brushes.

রয়্যাল অ্যাডন্স ফর এলিমেন্টর, এলিমেন্টর পেজ বিল্ডারকে ১৩০+ কাস্টম উইজেট দিয়ে সমৃদ্ধ করে। কোনো কোড ছাড়াই স্লাইডার, ক্যারোসেল, মেনু, পোস্ট গ্রিড, WooCommerce ডিসপ্লে, সোশ্যাল ফিড, চার্ট ইত্যাদি পেজে ড্র্যাগ করে যোগ করুন।

2. Installation & Activation২. ইন্সটলেশন ও একটিভেশন

Requirements: WordPress 6.7+, PHP 7.4+, Elementor (free or pro) installed first.

  1. Go to Plugins → Add New → Upload Plugin and upload the zip file.
  2. Click Activate Plugin. A "Royal Addons" menu appears in the sidebar.
  3. Go to Royal Addons → License and enter your license key to unlock all widgets.
  4. Open any page in Elementor and search "Royal" in the widget panel to find all widgets.

প্রয়োজনীয়: WordPress 6.7+, PHP 7.4+, এবং Elementor আগে ইন্সটল থাকতে হবে।

  1. Plugins → Add New → Upload Plugin থেকে zip ফাইল আপলোড করুন।
  2. Activate Plugin ক্লিক করুন। সাইডবারে "Royal Addons" মেনু দেখা যাবে।
  3. Royal Addons → License-এ গিয়ে লাইসেন্স কী দিন এবং সব উইজেট আনলক করুন।
  4. Elementor দিয়ে পেজ খুলুন এবং "Royal" লিখে সার্চ করুন।

4. Adding a Widget to a Page৪. পেজে একটি উইজেট যোগ করা

Example: Adding Royal Post Grid to your Blog pageউদাহরণ: Blog পেজে Royal Post Grid যোগ করা
  1. Step 1: Edit your Blog page with Elementor and click "+" to add a new section.ধাপ ১: Elementor দিয়ে Blog পেজ এডিট করুন এবং "+" ক্লিক করে নতুন সেকশন যোগ করুন।
  2. Step 2: In the widget panel, type "Royal Post Grid" and drag it into the section.ধাপ ২: উইজেট প্যানেলে "Royal Post Grid" লিখে সার্চ করুন এবং টেনে সেকশনে ছাড়ুন।
  3. Step 3: In settings, choose Post Type, number of columns, posts count, and Layout Style (Card/Classic/List).ধাপ ৩: সেটিংসে Post Type, কলাম সংখ্যা, পোস্ট সংখ্যা এবং Layout Style (Card/Classic/List) বেছে নিন।
  4. Step 4: Switch to the Style tab to customize colors, typography, and hover effects.ধাপ ৪: Style ট্যাবে গিয়ে রং, টাইপোগ্রাফি ও হোভার ইফেক্ট কাস্টমাইজ করুন।
  5. Step 5: Click Update. Your posts now display in a beautiful grid!ধাপ ৫: Update ক্লিক করুন। পোস্টগুলো সুন্দর গ্রিডে দেখাচ্ছে!

5. Custom CSS Feature৫. কাস্টম CSS ফিচার

Royal Addons includes a built-in Custom CSS editor for writing global CSS that applies across your entire site without editing theme files.

  • Go to Royal Addons → Custom CSS.
  • Write your CSS rules (e.g., button colors, font sizes, spacing) in the code editor.
  • Click Save Changes. CSS is automatically loaded on the front-end.
  • Tip: Use Browser Inspect to find the correct selector first.

রয়্যাল অ্যাডন্সে একটি বিল্ট-ইন Custom CSS এডিটর আছে। থিম ফাইল না ছুঁয়েই পুরো সাইটে গ্লোবাল CSS লেখা যায়।

  • Royal Addons → Custom CSS-এ যান।
  • কোড এডিটরে CSS নিয়ম লিখুন (বাটনের রং, ফন্ট সাইজ, স্পেসিং পরিবর্তন করতে)।
  • Save Changes ক্লিক করুন। CSS স্বয়ংক্রিয়ভাবে ফ্রন্ট-এন্ডে লোড হবে।
  • টিপস: প্রথমে ব্রাউজারের Inspect টুল দিয়ে সঠিক সিলেক্টর খুঁজুন।

6. Royal Template Library৬. রয়্যাল টেমপ্লেট লাইব্রেরি

Import ready-made, professionally designed page templates with a single click.

  1. Open any page in Elementor.
  2. Click the Folder icon (Templates library) in the Elementor toolbar.
  3. Switch to the "Royal Templates" tab, browse, and click "Insert".
  4. The template loads instantly. Replace demo text and images with your own content.

মাত্র একটি ক্লিকে রেডিমেড, প্রফেশনাল পেজ টেমপ্লেট ইম্পোর্ট করুন।

  1. Elementor দিয়ে যেকোনো পেজ খুলুন।
  2. Elementor টুলবারে ফোল্ডার আইকনে ক্লিক করুন।
  3. "Royal Templates" ট্যাবে যান, টেমপ্লেট বেছে "Insert" করুন।
  4. টেমপ্লেট লোড হলে ডেমো টেক্সট ও ছবি নিজের কন্টেন্ট দিয়ে বদলান।
