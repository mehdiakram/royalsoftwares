1. What is Royal Learn?১. রয়্যাল লার্ন কী?

Royal Learn is a complete Student Management System (SMS) built as a WordPress plugin. It helps educational institutes — coaching centers, computer training centers, madrasas, schools, and universities — manage their students, courses, batches, fees, attendance, and results all in one place.

No separate software is needed. Everything runs inside your existing WordPress website.

রয়্যাল লার্ন হলো একটি সম্পূর্ণ শিক্ষার্থী ব্যবস্থাপনা সিস্টেম (SMS), যা ওয়ার্ডপ্রেস প্লাগইন হিসেবে তৈরি। এটি শিক্ষাপ্রতিষ্ঠানগুলোকে — কোচিং সেন্টার, কম্পিউটার প্রশিক্ষণ কেন্দ্র, মাদ্রাসা, স্কুল এবং বিশ্ববিদ্যালয়গুলোকে — তাদের শিক্ষার্থী, কোর্স, ব্যাচ, ফি, উপস্থিতি এবং ফলাফল এক জায়গায় পরিচালনা করতে সাহায্য করে।

আলাদা কোনো সফটওয়্যার লাগবে না। সবকিছু আপনার বিদ্যমান ওয়ার্ডপ্রেস ওয়েবসাইটের ভেতর থেকেই চলে।

2. Installation & Setup২. ইন্সটলেশন ও সেটআপ

  1. Upload and activate the plugin via Plugins → Add New → Upload Plugin.
  2. The plugin automatically creates all required database tables.
  3. Go to Royal Learn → Settings to set your institute name, logo, academic year, and grading policy.
  4. Enter your license key under Royal Learn → License to activate all features.
  1. Plugins → Add New → Upload Plugin থেকে আপলোড করে প্লাগইন একটিভ করুন।
  2. প্লাগইন স্বয়ংক্রিয়ভাবে প্রয়োজনীয় সব ডেটাবেজ টেবিল তৈরি করে নেবে।
  3. Royal Learn → Settings-এ গিয়ে আপনার প্রতিষ্ঠানের নাম, লোগো, শিক্ষাবর্ষ এবং গ্রেডিং নীতি সেট করুন।
  4. Royal Learn → License-এ লাইসেন্স কী দিয়ে সব ফিচার একটিভ করুন।

3. Adding Students৩. শিক্ষার্থী যোগ করা

How to register a new studentনতুন শিক্ষার্থী নিবন্ধন করা
  1. Go to Royal Learn → Students → Add New.Royal Learn → Students → Add New-এ যান।
  2. Fill in: Full Name, Date of Birth, Gender, Phone, Email, Guardian Name, and Address.পূরণ করুন: পুরো নাম, জন্মতারিখ, লিঙ্গ, ফোন, ইমেইল, অভিভাবকের নাম এবং ঠিকানা।
  3. Upload the student's photo for the ID card.শিক্ষার্থীর ছবি আপলোড করুন (ID কার্ডের জন্য)।
  4. Click Save Student. The system auto-generates a unique Student ID.Save Student ক্লিক করুন। সিস্টেম স্বয়ংক্রিয়ভাবে একটি ইউনিক Student ID তৈরি করবে।

4. Courses & Batches৪. কোর্স ও ব্যাচ

Structure your teaching programs with Courses and organize them into time-based Batches.

ConceptWhat it isExample
CourseThe subject or program being taught"Complete Web Design", "Spoken English", "Class 6 Science"
BatchA group of students in the same course start & end period"Web Design – Batch 5 (March 2026 – June 2026)"

Go to Royal Learn → Courses to create courses, then create a batch under each course with its start date, end date, and monthly fee.

আপনার শিক্ষাক্রমকে কোর্স দিয়ে সাজান এবং সময়ভিত্তিক ব্যাচে বিভক্ত করুন।

ধারণাকীউদাহরণ
কোর্সযে বিষয় বা প্রোগ্রাম পড়ানো হচ্ছে"সম্পূর্ণ ওয়েব ডিজাইন", "স্পোকেন ইংলিশ", "৬ষ্ঠ শ্রেণির বিজ্ঞান"
ব্যাচএকই কোর্সে একটি নির্দিষ্ট সময়ে পড়া শিক্ষার্থীদের দল"ওয়েব ডিজাইন – ব্যাচ ৫ (মার্চ ২০২৬ – জুন ২০২৬)"

Royal Learn → Courses-এ গিয়ে কোর্স তৈরি করুন, তারপর প্রতিটি কোর্সের অধীনে শুরু/শেষ তারিখ ও মাসিক ফি সহ ব্যাচ তৈরি করুন।

5. Enrollment (Admitting Students to Batches)৫. ভর্তি প্রক্রিয়া (শিক্ষার্থীদের ব্যাচে যোগ করা)

Once a student and a batch exist, enroll the student into the batch.

  • Go to Royal Learn → Enrollments → Add New.
  • Select the Student, the Course, and the Batch.
  • Set the Enrollment Date and any initial discount.
  • Click Enroll. The student is now linked to the batch and fees can be collected.

শিক্ষার্থী ও ব্যাচ তৈরির পর শিক্ষার্থীকে ব্যাচে ভর্তি করুন।

  • Royal Learn → Enrollments → Add New-এ যান।
  • শিক্ষার্থী, কোর্স এবং ব্যাচ সিলেক্ট করুন।
  • ভর্তির তারিখ এবং কোনো ছাড় থাকলে সেট করুন।
  • Enroll ক্লিক করুন। শিক্ষার্থী এখন ব্যাচের সাথে যুক্ত এবং ফি সংগ্রহ করা যাবে।

6. Fee Collection৬. ফি সংগ্রহ

Collecting the monthly fee from a studentশিক্ষার্থীর মাসিক ফি সংগ্রহ করা
  1. Go to Royal Learn → Fees → Add Payment.Royal Learn → Fees → Add Payment-এ যান।
  2. Select the Student and the Month for which the fee is being paid.শিক্ষার্থী এবং যে মাসের ফি দেওয়া হচ্ছে তা সিলেক্ট করুন।
  3. Enter the Amount Paid and Payment Method. Apply any waiver if applicable.পরিশোধিত পরিমাণ ও পেমেন্ট মেথড লিখুন। প্রযোজ্য হলে ছাড় প্রয়োগ করুন।
  4. Click Save. A printable fee receipt is generated automatically.Save ক্লিক করুন। একটি প্রিন্টযোগ্য ফি রশিদ স্বয়ংক্রিয়ভাবে তৈরি হবে।

7. Attendance৭. উপস্থিতি

Take daily class attendance for each batch.

  • Go to Royal Learn → Attendance → Take Attendance.
  • Select the Batch and Date. A list of all enrolled students appears.
  • Mark each student as Present, Absent, or Late.
  • Click Submit to save. You can review the monthly attendance summary per student from Attendance Report.

প্রতিটি ব্যাচের দৈনিক ক্লাসের উপস্থিতি নিন।

  • Royal Learn → Attendance → Take Attendance-এ যান।
  • ব্যাচ এবং তারিখ সিলেক্ট করুন। সব ভর্তিকৃত শিক্ষার্থীদের তালিকা দেখাবে।
  • প্রতিটি শিক্ষার্থীকে উপস্থিত, অনুপস্থিত, বা দেরিতে এসেছে চিহ্নিত করুন।
  • Submit ক্লিক করে সংরক্ষণ করুন। Attendance Report থেকে প্রতি শিক্ষার্থীর মাসিক সারসংক্ষেপ দেখা যাবে।

8. Results & Grades৮. ফলাফল ও গ্রেড

Record exam marks and let the system calculate grades automatically.

  • Go to Royal Learn → Results → Add Result.
  • Select the Batch, Exam Type (Mid-term, Final), and Subject.
  • Enter marks for each student. The system applies your grading policy (A+, A, B, etc.) automatically based on the Settings you configured.
  • Publish the result. Students and guardians can see results on the portal.

পরীক্ষার নম্বর রেকর্ড করুন এবং সিস্টেম স্বয়ংক্রিয়ভাবে গ্রেড হিসাব করবে।

  • Royal Learn → Results → Add Result-এ যান।
  • ব্যাচ, পরীক্ষার ধরন (সাময়িক, বার্ষিক) এবং বিষয় সিলেক্ট করুন।
  • প্রতিটি শিক্ষার্থীর নম্বর দিন। সেটিংসে নির্ধারিত গ্রেডিং নীতি অনুযায়ী সিস্টেম স্বয়ংক্রিয়ভাবে গ্রেড (A+, A, B ইত্যাদি) প্রয়োগ করবে।
  • ফলাফল প্রকাশ করুন। শিক্ষার্থী ও অভিভাবকরা পোর্টালে ফলাফল দেখতে পাবেন।

  • Minimum Server Requirements:
    • WordPress 6.0+
    • PHP 8.0+
    • Sufficient storage and bandwidth for video hosting
  • ধরন (Type): ওয়েবসাইট / লার্নিং ম্যানেজমেন্ট সিস্টেম (WordPress)
