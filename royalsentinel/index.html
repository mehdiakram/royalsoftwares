1. What is Royal Sentinel?১. রয়্যাল সেন্টিনেল কী?

Royal Sentinel is an all-in-one security and performance optimization toolkit for WordPress. It protects your site from hackers, malware, brute-force attacks, and bad bots — while also improving performance with smart tweaks.

Key protection modules include: Login Protection, Two-Factor Authentication (2FA), Firewall, IP Blocking, Malware Scanner, Honeypot, Geoblocking, Security Headers, Session Management, Database Prefix Change, and Cloudflare integration.

রয়্যাল সেন্টিনেল হলো ওয়ার্ডপ্রেসের জন্য একটি অল-ইন-ওয়ান নিরাপত্তা ও পারফরম্যান্স অপটিমাইজেশন টুলকিট। এটি আপনার সাইটকে হ্যাকার, ম্যালওয়্যার, ব্রুট-ফোর্স আক্রমণ এবং খারাপ বট থেকে রক্ষা করে — এবং স্মার্ট টুইকসের মাধ্যমে পারফরম্যান্সও উন্নত করে।

মূল সুরক্ষা মডিউলগুলো: লগইন সুরক্ষা, টু-ফ্যাক্টর অথেন্টিকেশন (2FA), ফায়ারওয়াল, IP ব্লকিং, ম্যালওয়্যার স্ক্যানার, হানিপট, জিওব্লকিং, সিকিউরিটি হেডার, সেশন ম্যানেজমেন্ট, ডেটাবেজ প্রিফিক্স পরিবর্তন এবং Cloudflare ইন্টিগ্রেশন।

2. Installation২. ইন্সটলেশন

  1. Upload and activate via Plugins → Add New → Upload Plugin.
  2. On activation, the plugin creates its own bad IPs database table automatically.
  3. Go to Royal Sentinel in the WordPress sidebar to see all security modules.
  4. Enter your license key under Royal Sentinel → License to enable all features.
  5. Tip: If you are ever accidentally locked out, use the emergency unlock script included in the plugin folder.
  1. Plugins → Add New → Upload Plugin থেকে আপলোড করে একটিভ করুন।
  2. একটিভেশনের সময় প্লাগইন স্বয়ংক্রিয়ভাবে একটি bad IPs ডেটাবেজ টেবিল তৈরি করে।
  3. ওয়ার্ডপ্রেস সাইডবারে Royal Sentinel-এ গিয়ে সব সিকিউরিটি মডিউল দেখুন।
  4. Royal Sentinel → License-এ লাইসেন্স কী দিয়ে সব ফিচার চালু করুন।
  5. টিপস: কখনো ভুলক্রমে লক আউট হলে প্লাগইন ফোল্ডারে থাকা emergency unlock স্ক্রিপ্ট ব্যবহার করুন।

3. Login Protection৩. লগইন সুরক্ষা

Brute-force attacks try thousands of password combinations against your login page. Royal Sentinel stops this with multiple layers:

  • Limit Login Attempts: Set the max failed attempts (e.g., 5) before an IP is temporarily banned. Configure the lockout duration (e.g., 30 minutes).
  • Hide Login URL: Change your login page from the default /wp-admin to a custom URL (e.g., /my-login) so bots can't even find it.
  • Login Extras: Disable the "password reset" link, add a CAPTCHA to the login form, and show a custom error message that doesn't reveal whether username or password was wrong.

ব্রুট-ফোর্স আক্রমণ আপনার লগইন পেজে হাজার হাজার পাসওয়ার্ড কম্বিনেশন চেষ্টা করে। রয়্যাল সেন্টিনেল একাধিক স্তরে এটি থামায়:

  • Login Attempt সীমিত করা: একটি IP থেকে সর্বোচ্চ ব্যর্থ চেষ্টার সংখ্যা (যেমন: ৫) এবং ব্লকের দৈর্ঘ্য (যেমন: ৩০ মিনিট) সেট করুন।
  • লগইন URL লুকানো: ডিফল্ট /wp-admin থেকে একটি কাস্টম URL-এ (যেমন: /my-login) পরিবর্তন করুন যাতে বটরা এটি খুঁজে না পায়।
  • Login Extras: "পাসওয়ার্ড রিসেট" লিঙ্ক অক্ষম করুন, লগইন ফর্মে CAPTCHA যোগ করুন এবং ইউজারনেম বা পাসওয়ার্ড কোনটা ভুল তা প্রকাশ না করে কাস্টম এরর মেসেজ দেখান।

4. Two-Factor Authentication (2FA)৪. টু-ফ্যাক্টর অথেন্টিকেশন (2FA)

Setting up 2FA for your admin accountঅ্যাডমিন একাউন্টের জন্য 2FA চালু করা
  1. Go to Royal Sentinel → 2FA and enable 2FA for the Administrator role.Royal Sentinel → 2FA-এ গিয়ে Administrator রোলের জন্য 2FA চালু করুন।
  2. Log out and log back in. After entering your username and password, a 6-digit code will be sent to your email.লগ আউট করে আবার লগ ইন করুন। ইউজারনেম ও পাসওয়ার্ড দেওয়ার পর একটি ৬ সংখ্যার কোড আপনার ইমেইলে পাঠানো হবে।
  3. Enter the code in the verification field to complete the login.লগইন সম্পন্ন করতে ভেরিফিকেশন ফিল্ডে কোডটি দিন।
  4. Result: Even if a hacker knows your password, they still cannot log in without access to your email.ফলাফল: কোনো হ্যাকার আপনার পাসওয়ার্ড জানলেও আপনার ইমেইল অ্যাক্সেস ছাড়া লগইন করতে পারবে না।

5. Firewall & IP Blocking৫. ফায়ারওয়াল ও IP ব্লকিং

Royal Sentinel's firewall module intercepts malicious requests before they reach WordPress.

  • Auto Block: After a configured number of failed login attempts, the offending IP is automatically added to the bad IPs list and blocked.
  • Manual Block: Go to Royal Sentinel → Bad IPs and manually enter any IP address to block instantly.
  • IP Whitelist: Add your own static IP so you are never accidentally blocked.
  • Geoblocking: Block all traffic from specific countries entirely. Useful if your site only serves one region.
  • Real-time Defense: Monitors for common attack patterns (SQL injection, XSS, file inclusion) and blocks attackers in real time.

রয়্যাল সেন্টিনেলের ফায়ারওয়াল মডিউল ক্ষতিকর অনুরোধগুলো ওয়ার্ডপ্রেসে পৌঁছানোর আগেই আটকায়।

  • অটো ব্লক: নির্ধারিত সংখ্যক ব্যর্থ লগইন ক্যালকুলেশনের পর আক্রমণকারী IP স্বয়ংক্রিয়ভাবে bad IPs তালিকায় যোগ হয় এবং ব্লক হয়।
  • ম্যানুয়াল ব্লক: Royal Sentinel → Bad IPs-এ গিয়ে যেকোনো IP ঠিকানা সাথে সাথে ব্লক করুন।
  • IP হোয়াইটলিস্ট: আপনার নিজের স্ট্যাটিক IP যোগ করুন যাতে আপনি কখনো ভুলক্রমে ব্লক না হন।
  • জিওব্লকিং: নির্দিষ্ট দেশের সব ট্র্যাফিক সম্পূর্ণভাবে ব্লক করুন। আপনার সাইট যদি একটি অঞ্চলেই সেবা দেয় তাহলে কার্যকর।
  • Real-time Defense: সাধারণ আক্রমণের ধরন (SQL ইনজেকশন, XSS) পর্যবেক্ষণ করে এবং রিয়েল-টাইমে আক্রমণকারীদের ব্লক করে।

6. Malware Scanner৬. ম্যালওয়্যার স্ক্যানার

The scanner analyzes your WordPress files for malicious code, backdoors, and known vulnerabilities.

  1. Go to Royal Sentinel → Malware Scanner.
  2. Click Start Scan. The system scans all WordPress core files, themes, and plugins.
  3. Suspicious files are listed with their path and threat type. Review each finding.
  4. The Vulnerability Scanner additionally checks if your installed plugins and themes have known CVE vulnerabilities reported in public security databases.

স্ক্যানার আপনার ওয়ার্ডপ্রেস ফাইলগুলো ক্ষতিকর কোড, ব্যাকডোর এবং পরিচিত দুর্বলতার জন্য বিশ্লেষণ করে।

  1. Royal Sentinel → Malware Scanner-এ যান।
  2. Start Scan ক্লিক করুন। সিস্টেম সব ওয়ার্ডপ্রেস কোর ফাইল, থিম এবং প্লাগইন স্ক্যান করবে।
  3. সন্দেহজনক ফাইলগুলো তাদের পাথ ও হুমকির ধরন সহ তালিকাবদ্ধ হবে। প্রতিটি ফলাফল পর্যালোচনা করুন।
  4. Vulnerability Scanner অতিরিক্তভাবে চেক করে যে আপনার ইন্সটল করা প্লাগইন ও থিমে পাবলিক সিকিউরিটি ডেটাবেজে রিপোর্ট করা পরিচিত CVE দুর্বলতা আছে কিনা।

7. Backup৭. ব্যাকআপ

Before making big changes to your site, always create a backup. Royal Sentinel's backup module lets you:

  • Create an on-demand database backup or full-site backup (files + database) with one click.
  • Schedule automatic backups (daily, weekly) so you always have a recent copy.
  • Download backup archives directly to your computer for off-site storage.
  • Restore from a previous backup if something goes wrong after an update.

সাইটে বড় পরিবর্তনের আগে সবসময় ব্যাকআপ তৈরি করুন। রয়্যাল সেন্টিনেলের ব্যাকআপ মডিউল দিয়ে:

  • এক ক্লিকে অন-ডিমান্ড ডেটাবেজ ব্যাকআপ বা ফুল-সাইট ব্যাকআপ (ফাইল + ডেটাবেজ) তৈরি করুন।
  • স্বয়ংক্রিয় ব্যাকআপ শিডিউল করুন (দৈনিক, সাপ্তাহিক) যাতে সবসময় একটি সাম্প্রতিক কপি থাকে।
  • অফ-সাইট স্টোরেজের জন্য সরাসরি আপনার কম্পিউটারে ব্যাকআপ আর্কাইভ ডাউনলোড করুন।
  • কোনো আপডেটের পর সমস্যা হলে পূর্ববর্তী ব্যাকআপ থেকে পুনরুদ্ধার করুন।

9. Cloudflare Integration৯. Cloudflare ইন্টিগ্রেশন

Royal Sentinel can communicate directly with your Cloudflare account to block bad IPs at the network level — before they even reach your server.

  • Go to Royal Sentinel → Cloudflare and enter your Cloudflare API Token and Zone ID.
  • Enable CF Sync. When Royal Sentinel blocks an IP locally, it automatically pushes the block to Cloudflare's firewall rules as well.
  • This provides double-layer protection: local WordPress block + Cloudflare edge block.
  • You can also manage Cloudflare firewall rules (allow/block by country or IP range) directly from the Royal Sentinel dashboard.

রয়্যাল সেন্টিনেল আপনার Cloudflare একাউন্টের সাথে সরাসরি যোগাযোগ করে নেটওয়ার্ক স্তরে খারাপ IP ব্লক করতে পারে — এমনকি আপনার সার্ভারে পৌঁছানোর আগেই।

  • Royal Sentinel → Cloudflare-এ গিয়ে আপনার Cloudflare API Token এবং Zone ID দিন।
  • CF Sync চালু করুন। রয়্যাল সেন্টিনেল যখন স্থানীয়ভাবে কোনো IP ব্লক করে, এটি স্বয়ংক্রিয়ভাবে Cloudflare-এর ফায়ারওয়াল নিয়মেও ব্লকটি পুশ করে।
  • এটি ডাবল-লেয়ার সুরক্ষা দেয়: স্থানীয় ওয়ার্ডপ্রেস ব্লক + Cloudflare এজ ব্লক।
  • রয়্যাল সেন্টিনেল ড্যাশবোর্ড থেকেই সরাসরি Cloudflare ফায়ারওয়াল নিয়ম পরিচালনা করা যায়।
